  2 | 
  3 | ## 1. 目的
  4 | 指导用户在本地或线上环境运行 SproutBridge，并完成课程浏览与资源访问。
  5 | 
  6 | ## 2. 安装与运行
  7 | - 前置条件：安装 Node.js 18+ 与 npm。
  8 | - 步骤：
  9 | ```
 10 | npm install
 11 | npm run dev
 12 | # 浏览器打开 http://localhost:3000/
 13 | ```
 14 | - 生产部署：
 15 | ```
 16 | npm run build
 17 | npm run start
 18 | ```
 19 | 
 20 | ## 3. 功能操作指南
 21 | - 首页导航：按大类（语言训练、兴趣与素养、情感与心理、安全与生活、数据与激励）浏览课程。
 22 | - 课程详情：点击课程卡片进入详情页，查看简介、时长、提取码与资源链接。
 23 | - 提取码复制：在详情页点击复制按钮，将提取码复制到剪贴板。
 24 | - 跳转百度网盘：点击课程资源链接，跳转至网盘页面（请注意网络与账号登录）。
 25 | - 更新日志：在「更新日志」页面查看历史版本与变化。
 26 | - 帮助页面：查看观看与操作常见问题。
 27 | 
 28 | ## 4. 常见问题（FAQ）
 29 | - 无法访问资源链接：检查网络、浏览器拦截与百度网盘登录状态。
 30 | - 页面样式错乱：刷新缓存；确保浏览器版本较新；如为移动端，尝试横竖屏切换。
 31 | - 本地启动失败：确认 Node 版本；重新执行 `npm install`；查看终端报错并按提示修复。
 32 | 
 33 | ## 5. 使用建议
 34 | - 建议使用桌面浏览器获得最佳体验，移动端已适配常见断点。
 35 | - 如在学校或单位网络环境使用，需允许外部链接访问。
